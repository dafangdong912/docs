The SOS helper page needs your GPS position to generate a useful distress message. Without coordinates, your contact receives no location data and cannot direct help to you. Granting location access to Safari takes under ten seconds — do it now, before you ever need it.

## Grant Permission on First Visit

<Steps>
  <Step title="Open the SOS page in iPhone Safari">
    Navigate to the helper page URL using the QR code or by typing the address. Confirm the blinking red status indicator is visible.
  </Step>
  <Step title="Tap '获取我的当前位置'">
    Tap the **Get Location** button. This is the only action that triggers a location request — the page calls `navigator.geolocation.getCurrentPosition` at that moment and never requests your position automatically.
  </Step>
  <Step title="Respond to the permission dialog">
    Safari displays a prompt asking whether to share your location. Tap **Allow Once** to grant access for this session, or **Allow While Using** to grant access whenever the page is open.
  </Step>
  <Step title="Wait for the coordinates to appear">
    Within a few seconds the coordinates display fills with your latitude and longitude (each to six decimal places), accuracy in metres, and a timestamp. Your SOS message is now ready to send.
  </Step>
</Steps>

<Tip>
  Choose **Allow While Using** rather than **Allow Once**. With this setting Safari does not prompt you again each time you tap **获取我的当前位置**, which matters when you need to act quickly under stress.
</Tip>

## Re-Enable Location If Previously Denied

If you tapped **Don't Allow** at the Safari prompt, iOS blocks all future location requests from that page until you override the setting manually. Use one of these paths to restore access:

**Path A — Location Services settings:**
1. Open the iPhone **Settings** app.
2. Go to **Privacy & Security → Location Services**.
3. Scroll down and tap **Safari Websites**.
4. Set the permission to **While Using the App**.

**Path B — Safari settings:**
1. Open the iPhone **Settings** app.
2. Scroll down and tap **Safari**.
3. Tap **Location**.
4. Set it to **Ask** or **Allow**.

After changing either setting, return to the SOS page and tap **获取我的当前位置** again.

<Warning>
  If location access is denied, the SOS helper page displays an error and cannot generate an SOS message. The coordinates, Apple Maps link, and Google Maps link are all derived from your GPS position — none of them appear without it. Resolve the permission issue before an emergency arises.
</Warning>

## GPS Parameters the Page Uses

When you tap **获取我的当前位置**, the page calls `navigator.geolocation.getCurrentPosition` with the following options:

| Parameter | Value | Effect |
|---|---|---|
| `enableHighAccuracy` | `true` | Instructs iOS to use GPS hardware rather than Wi-Fi or cell-tower triangulation alone, producing the most precise fix available. |
| `timeout` | `15000` ms | If iOS cannot return a position within 15 seconds, the page reports a **timeout error** (error code `3`) and invites you to try again. |
| `maximumAge` | `0` | Forces a fresh position fix every time you tap the button. Cached positions are never used. |

## Location Error Codes

If something goes wrong the page displays one of three error messages:

<Accordion title="Error 1 — Permission denied">
  iOS returned `PERMISSION_DENIED` (code `1`). Location Services is blocked for Safari or for this page. Follow the steps in **Re-Enable Location If Previously Denied** above to restore access.
</Accordion>

<Accordion title="Error 2 — Position unavailable">
  iOS returned `POSITION_UNAVAILABLE` (code `2`). The device could not determine your position — this usually happens deep indoors or in a basement with no GPS signal. Move to a window or step outdoors, then tap **获取我的当前位置** again.
</Accordion>

<Accordion title="Error 3 — Timeout">
  iOS returned `TIMEOUT` (code `3`). A position fix was not obtained within the 15-second window. This can happen in areas with a weak satellite signal. Wait a few seconds, then tap **获取我的当前位置** again. Moving closer to a window improves acquisition speed.
</Accordion>

## Understanding Accuracy

<Info>
  With `enableHighAccuracy: true`, iOS GPS typically achieves an accuracy of **10–30 metres** outdoors under a clear sky. The accuracy value is included in your SOS message so your contact knows how precise the fix is.
</Info>

Use these tips to get the best possible reading:

- **Go outdoors or stand near a window.** GPS signals are blocked by thick walls, basements, and dense building materials. Open sky gives the fastest and most accurate fix.
- **Wait and retry if accuracy is high.** If the coordinates display shows an accuracy greater than 100 metres, wait 5–10 seconds for the GPS receiver to refine its estimate, then tap **获取我的当前位置** again. The page always uses the most recently captured position.
- **Keep the screen on.** iOS may throttle location hardware when the display sleeps. Tap the screen to keep it active while the fix is refining.
